O que é isso de pensamento computacional?

Definição

É habilidade de pensar de forma lógica e organizada para resolver problemas - decompor em etapas simples para que consiga encontrar soluções eficientes

Quais são as 4 etapas do pensamento computacional?

  1. Decomposição

    2. Pega um problema grande e divide ele em partes menores e mais fáceis de resolver

  2. Reconhcimento de padrões

    3. É tipo encontrar semelhanças em problmemas que você já resolveu ou que já foram resolvidos

  3. Abstração

    4. Tira aquiles detalhes desnecessários e foca no que realmente importa para resolver os problemas

  4. Algoritmo

    5. Estrutura um passo a passo com instruções claras de forma objetiva para resolver o problema
